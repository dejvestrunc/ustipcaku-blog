# Ústipčáků Wine Blog — Design Spec

## Overview
Static wine blog at `blog.ustipcaku.cz` that visually matches the existing eshop at `ustipcaku.cz`. AI-generated daily articles about wine. Deployed via Docker in Coolify.

## Architecture

### Static Site (Astro)
- **Framework:** Astro with Markdown content collections
- **Content:** Articles stored as `.md` files in `src/content/articles/`
- **Output:** Static HTML/CSS/JS served by Nginx

### Article Generator (Node.js script)
- Standalone script on VPS (alongside Lumír)
- Daily cron job calls Claude API
- Generates Markdown with frontmatter (title, description, category, date, productLinks)
- Commits to git repo, pushes → Coolify auto-redeploy via webhook

### Deployment (Docker + Coolify)
- Multi-stage Dockerfile: Node (Astro build) → Nginx (serve `dist/`)
- Coolify pointed at `blog.ustipcaku.cz`
- Auto-redeploy on git push

## Visual Design

### Colors
- Primary red: `#A9010C` (hover: `#d93621`)
- Golden accent: `#FFB200` (hover: `#DE8604`)
- Text: `#333`
- Background: `#ffffff`

### Typography
- Headings: "Exo 2" (weights: 700, 900)
- Body: "Source Sans 3" (weights: 300, 400, 700)
- Google Fonts loaded

### Header
- Logo "U Štipčáků" (links to eshop ustipcaku.cz)
- Navigation: Články (all), category links
- CTA button "Koupit víno" → ustipcaku.cz (golden #FFB200, prominent)
- Responsive hamburger menu on mobile

### Footer
- Links to: eshop, degustace, ubytování nad sklepem
- Copyright © U Štipčáků
- Matching eshop footer style

## Pages

### Homepage (`/`)
- Hero section with blog title and short description
- Article listing as cards: thumbnail image, title, excerpt, date, category badge
- Paginated or infinite scroll (start with pagination)
- Category filter chips at top

### Article Page (`/clanky/[slug]`)
- Full article text with rich formatting
- Article metadata: date, category, author (AI-generated note)
- Sidebar/inline CTA blocks linking to specific products on ustipcaku.cz
- Related articles at bottom (same category)

### Category Pages (`/kategorie/[category]`)
- Filtered article listing
- Categories by wine type: Bílé víno, Červené víno, Rosé, Frizzante a šumivé
- Categories by topic: Párování s jídlem, Vinařské regiony, Tipy a triky

## Article Generator

### Tech
- Node.js script (`scripts/generate-article.js`)
- Uses `@anthropic-ai/sdk`
- Claude API with wine-focused system prompt

### Prompt Strategy
- Rotating topics across categories
- References real wine varieties and regions (Moravia focus — U Štipčáků is a Moravian winery)
- Includes product link suggestions from a maintained product list
- Generates SEO-friendly title, description, and content
- Czech language

### Cron Setup
- Runs on VPS alongside Lumír
- Daily at configurable time (e.g., 6:00 AM)
- `cd /path/to/repo && node scripts/generate-article.js && git add . && git commit -m "article: [title]" && git push`

## SEO Considerations
- Clean URLs (`/clanky/jak-parovat-ryzlink`)
- Meta tags (title, description, og:image)
- Sitemap.xml auto-generated by Astro
- robots.txt allowing crawling
- Internal links to eshop for domain authority sharing
